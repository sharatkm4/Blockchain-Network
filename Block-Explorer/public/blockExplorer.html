Block Explorer App

Welcome to the Block Explorer App using Test Blockchain network as part of Kingsland University.
  • The Block Explorer is an app that provides info about the blockchain network.
  • Hover on the below hyper links to get more info.

Node URL:

General Info and Debug

  • GET /info
  • GET /debug
  • GET /debug/reset-chain

Blocks

  • GET /blocks
  • GET /blocks/:index

Transactions

  • GET /transactions/pending
  • GET /transactions/confirmed
  • GET /transactions/{hash}
  • POST /transactions/send

Balances and Addresses

  • GET /balances
  • GET /address/{address}/transactions
  • GET /address/{address}/balance

Mining

  • GET /mining/get-mining-job/{minerAddress}
  • POST /mining/submit-mined-block
  • GET /debug/mine/{minerAddress}/{difficulty}

Peers

  • GET /peers
  • POST /peers/connect
  • POST /peers/notify-new-block
